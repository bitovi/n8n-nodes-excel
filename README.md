# @bitovi/n8n-nodes-excel

This repo contains [N8N](https://n8n.io/) node to work with Excel files.

## Installation

- Make sure to allow community nodes with `N8N_COMMUNITY_PACKAGES_ENABLED=true`
- Once logged in to your N8N web UI, go to `/settings/community-nodes` and type `@bitovi/n8n-nodes-excel`
